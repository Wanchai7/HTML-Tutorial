Introduction to HTML

- HTML stands for Hyper Text Markup Language
- HTML is the standard markup language for creating Web pages
- HTML describes the structure of a Web page
- HTML consists of a series of elements
- HTML elements tell the browser how to display the content
- HTML elements are represented by tags

HTML Document Structure

An HTML document has the following basic structure...

คำอธิบาย:
<!DOCTYPE html>: ระบุว่าเป็นเอกสาร HTML5
<html>: เป็นรากของเอกสาร HTML ทั้งหมด
<head>: ส่วนที่ใช้เก็บข้อมูลเมตา เช่น <title>
<body>: ส่วนที่จะแสดงผลบนหน้าเว็บ

แท็ก (Tags):
    - <h1> แท็กเริ่มต้น
    - </h1> แท็กปิด

ตัวอย่าง: <h1>Hello, World!</h1>

Text Formatting

Creating Links

Workshop

HTML links are defined with the <a> tag...

Syntax: <a href="url" target="_blank">Link Text</a>

Attributes:
- href: ลิงก์ปลายทาง
- target="_blank": เปิดหน้าต่างใหม่
- mailto: สำหรับส่งอีเมล

Lists (Ordered and Unordered)

Workshop

HTML lists are defined with the <ul>, <ol>, and <li> tags.

Unordered List (ไม่มีลำดับ):
<ul>
  <li>Coffee</li>
  <li>Tea</li>
</ul>

Ordered List (มีตัวเลข):
<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>
