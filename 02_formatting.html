Workshop: HTML Text Formatting

ฝึกใช้แท็กรูปแบบข้อความ (Formatting Tags) เพื่อสร้างเนื้อหาที่อ่านง่ายและเป็นระเบียบ

1. สิ่งที่ต้องรู้ก่อนเริ่มทำ Workshop

HTML มีแท็กสำหรับจัดรูปแบบข้อความ ทำให้เนื้อหาดูเด่น อ่านง่าย และสื่อความหมายได้ดีขึ้น

  • <b> : ตัวหนา (เน้นแบบทั่วไป)
  • <strong> : ตัวหนาแบบ "สำคัญ" เชิงความหมาย (Semantic Importance)
  • <i> : ตัวเอียง (ใช้กับศัพท์เฉพาะหรือชื่อเรื่อง)
  • <u> : ขีดเส้นใต้ (Underline)
  • <mark> : ไฮไลต์ข้อความ
  • <code> : แสดงข้อความเป็นรูปแบบโค้ด
  • <small> : ข้อความขนาดเล็ก (เช่น หมายเหตุ)
  • <del> : ข้อความขีดฆ่า (เช่น ราคาก่อนลด)
  • <sub> : ตัวห้อย (Subscript เช่น H2O)
  • <sup> : ตัวยก (Superscript เช่น X2)

2. ภารกิจสำหรับนักศึกษาต้องทำ

ให้ค้นหาเนื้อหาเกี่ยวกับ “เทคโนโลยีคอมพิวเตอร์สมัยใหม่” โดยต้องมีองค์ประกอบดังนี้:

1. หัวข้อ <h1> จำนวน 1 อัน
2. ย่อหน้า <p> อย่างน้อย 3 ย่อหน้า
3. ใช้แท็ก Formatting อย่างน้อย 5 ประเภท
4. ข้อความไหนมีการจัด Formatting ให้ใส่สีข้อความ เป็นสีชมพู (Inline Style)

3. เทคโนโลยีคอมพิวเตอร์สมัยใหม่ (Quantum Computing)

ในยุคปัจจุบัน เทคโนโลยีคอมพิวเตอร์กำลังก้าวเข้าสู่ยุคใหม่ด้วย Quantum Computing (คอมพิวเตอร์ควอนตัม) ซึ่งมีพลังการประมวลผลที่ รวดเร็วกว่าซูเปอร์คอมพิวเตอร์ในปัจจุบันมหาศาล โดยทำงานบนพื้นฐานของ Qubits (Quantum Bits) แทนที่จะเป็น Bits ธรรมดา เทคโนโลยีนี้จะเข้ามาเปลี่ยนโลกในหลายด้าน แต่ก็อาจทำให้ระบบความปลอดภัยเดิม กลายเป็นสิ่งที่ล้าสมัย ไปในทันที จึงถือเป็น นวัตกรรมที่สำคัญที่สุด แห่งศตวรรษนี้

นอกจากนี้ การมาถึงของ AI ยังช่วยเสริมศักยภาพให้การคำนวณซับซ้อนขึ้นไปอีกขั้น ทำให้มนุษย์สามารถแก้ปัญหาที่ไม่เคยแก้ได้มาก่อน

เตรียมพร้อมรับมือกับการเปลี่ยนแปลงครั้งใหญ่นี้ได้เลย!
